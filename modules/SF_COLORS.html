SF_COLORS

Description

SciFortran module for color definitions and manipulation

Quick access

Types:

rgb_color

Routines:

add_colors(), assignment(=)(), dot_scalar_colors(), equal_colors(), operator(*)(), operator(+)(), operator(-)(), operator(.dot.)(), pick_color(), rgb(), scalar_left_color(), scalar_right_color(), subtract_colors()

Types

type  sf_colors/rgb_color

A custom type for color definitions in rgb form. A large set of predefinite colors are provided. Artithmetic operators are overloaded to manipulate colors

Type fields:

  • r [integer] – Red component in the range [0:255]

  • g [integer] – Green component in the range [0:255]

  • b [integer] – Blue component in the range [0:255]
