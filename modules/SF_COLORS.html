type  sf_colors/rgb_colorÔÉÅ

A custom type for color definitions in rgb form. A large set of predefinite colors are provided. Artithmetic operators are overloaded to manipulate colors

Type fields:

  • r [integer] ‚Äì Red component in the range [0:255]

  • g [integer] ‚Äì Green component in the range [0:255]

  • b [integer] ‚Äì Blue component in the range [0:255]
