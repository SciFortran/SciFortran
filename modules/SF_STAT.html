SF_STAT

Description

SciFortran module for statistical estimators and visualization

Quick access

Types:

histogram, pdf_kernel, pdf_kernel_2d

Routines:

get_covariance(), get_curt(), get_mean(), get_moments(), get_sd(), get_skew(), get_var(), histogram_accumulate(), histogram_allocate(), histogram_deallocate(), histogram_get_range(), histogram_get_value(), histogram_print(), histogram_reset(), histogram_set_range_uniform(), pdf_accumulate(), pdf_allocate(), pdf_curt(), pdf_deallocate(), pdf_mean(), pdf_moment(), pdf_normalize(), pdf_print(), pdf_print_moments(), pdf_push_sigma(), pdf_read(), pdf_save(), pdf_sdev(), pdf_set_range(), pdf_sigma(), pdf_skew(), pdf_var()

Used modules

  • sf_arrays: SciFortran module for array creation and manipulation

    • linspace()

  • sf_integrate: SciFortran module for function integration

    • simps()

  • sf_iotools: SciFortran module for reading and writing data

    • free_unit()

    • splot3d()

  • sf_linalg: SciFortran module for linear algebra

    • det()

    • inv()
